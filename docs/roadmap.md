# Roadmap

## Phase 0: Desktop MVP

- Transparent click-through overlay.
- Settings window.
- OpenAI-compatible provider with LM Studio defaults.
- Primary screen capture.
- Manual generation and timed generation.
- Local JSON config.

## Phase 1: Trust and Control

- Active app detection and blacklist enforcement.
- Capture preview with one-click redaction.
- Text-only OCR mode.
- Tray menu state polish.
- Provider connection test.

## Phase 2: Better Companionship

- Character profiles.
- Short local memory.
- Relevance scoring and duplicate suppression.
- Per-app behavior presets.
- Import/export config.

## Phase 3: Packaging

- Signed Windows installer.
- Portable ZIP build.
- Auto-update channel controlled by the user.
- macOS overlay validation.
